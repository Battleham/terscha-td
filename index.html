Platform Defense Prototype

Skyhold Rampart

Hold the sanctum against winged invaders. Run the battlements, cast spells, deploy defenders, and revive the fallen before the line breaks.

Controls

  • AD Move
  • Shift Run
  • W or Space Jump
  • Mouse 1 Arc strike
  • Mouse 2 Cast bolt
  • 123 Select defender
  • F Place defender (instant in prep, 2s cast in wave)
  • Tab Toggle placement mode
  • R Revive one defender, hold all in prep
  • Esc Pause / resume
